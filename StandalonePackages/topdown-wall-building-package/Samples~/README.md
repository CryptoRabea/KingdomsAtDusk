# Top-Down Wall Building System - Samples

This directory contains example scenes and scripts demonstrating how to use the wall building system.

## Planned Samples

### Sample 1: Basic Wall Placement
- Simple scene with flat terrain
- Pre-configured WallPlacementController
- UI button to start wall placement
- Resource manager implementation
- Wall data assets (wood, stone)

### Sample 2: Advanced Features
- Complex terrain
- Multiple wall types
- Resource cost preview UI
- Wall snapping demonstration
- Loop closure examples

### Sample 3: NavMesh Integration
- Walls blocking AI units
- NavMesh carving
- WallStairs for traversable walls
- Unit pathfinding demonstration

## How to Import Samples

1. Open Unity Package Manager
2. Find "Top-Down Wall Building System"
3. Expand "Samples" section
4. Click "Import" next to desired sample

## Implementation Note

Samples are not included in the initial release. They will be added in future updates based on user feedback.

## Quick Start Without Samples

See the main README.md for a complete quick start guide that will get you up and running without needing the sample scenes.
